# BusBuddy

BusBuddy is a shuttle management system with:
- **Frontend**: Vue.js
- **Backend**: Spring Boot

## How to run
### Backend
1. Open the `backend` folder in your IDE.
2. Run the Spring Boot application.

### Frontend
1. Open the `frontend` folder.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run serve`
